# GitSeeker GitHub user search web app

A responsive web app where you can search GitHub for developers by username and see a selection of the user's profile details and most recent repos. Integrated with the GitHub users API, built with React and styled with Tailwind CSS.

Design and functionality inspiration drawn from the [GitHub user search app challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/github-user-search-app-Q09YOgaH6).

## Overview

### The challenge

Users should be able to:

- Search for GitHub users by their username
- See relevant user information based on their search
- Switch between light and dark themes
- View the optimal layout for the app depending on their device's screen size
- See hover states for all interactive elements on the page
- Have correct color scheme chosen based on system defaults
- Manually switch between light and dark themes
